# PyQt6 FPS 准星工具 v1.0.0

一款专为 Windows 设计的 FPS 游戏准星工具，提供 200 个预设准星样式和 20 个专业主题，支持全屏/窗口化游戏。

## 🚀 主要特性

- ✅ **200 个准星预设** - 自动生成，覆盖所有主流 FPS 样式
- ✅ **20 个专业主题** - 精心设计的配色方案，满足不同审美需求
- ✅ **Windows 全屏/窗口化游戏通用** - 完美适配各种游戏模式
- ✅ **穿透点击不影响操作** - 点击穿透功能，不影响游戏操作
- ✅ **纯覆盖层，不读写游戏内存** - 100% 安全，不会被检测
- ✅ **可调颜色、大小、粗细** - 自定义准星外观
- ✅ **全局快捷键** - 游戏内无需切窗口即可控制
- ✅ **系统托盘集成** - 后台运行，方便管理
- ✅ **配置保存** - 自动保存用户设置

## 🛠️ 安装方法

### 方法一：直接使用（推荐）
1. 下载 `FPS_Crosshair_Tool.exe`
2. 双击运行即可使用
3. 无需安装 Python 环境

### 方法二：从源码运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

### 方法三：自行打包

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 构建可执行文件：
```bash
python build.py
```

3. 使用生成的 `dist/FPS_Crosshair_Tool.exe`

## 📖 使用说明

### 界面操作

1. **选择预设** - 从下拉菜单选择 200 个预设之一
2. **调整颜色** - 点击"选择颜色"按钮自定义颜色
3. **调整大小** - 拖动大小滑块（5-50像素）
4. **调整粗细** - 拖动粗细滑块（1-10像素）
5. **选择主题** - 从主题下拉菜单选择喜欢的配色方案
6. **显示/隐藏** - 点击按钮或使用快捷键
7. **点击穿透** - 勾选后鼠标点击穿透准星

### 快捷键

- `F6` - 显示/隐藏准星
- `F7` - 切换下一个预设
- `F8` - 切换上一个预设
- `Ctrl+Q` - 退出程序

## 🔍 故障排除

### 常见问题

1. **快捷键不响应**
   - 以管理员身份运行程序
   - 检查是否有其他程序占用快捷键
   - 关闭其他可能的冲突程序

2. **准星不显示**
   - 确保游戏不是独占全屏模式
   - 尝试窗口化或无边框全屏
   - 检查是否启用了"显示准星"选项

3. **打包失败**
   - 确保 PyInstaller 版本正确
   - 检查依赖是否完整安装
   - 尝试在虚拟环境中安装

4. **主题切换无效**
   - 重启程序
   - 检查配置文件是否损坏
   - 重新选择主题

### 权限要求

- 建议以管理员身份运行
- 需要键盘监听权限
- 需要窗口管理权限
- 部分杀毒软件可能需要添加白名单
