# PyQt6 FPS 准星工具

一款专为 Windows 设计的 FPS 游戏准星工具，提供 200 个预设准星样式，支持全屏/窗口化游戏。

## 主要特性

- ✅ **200 个准星预设** - 自动生成，覆盖所有主流 FPS 样式
- ✅ **Windows 全屏/窗口化游戏通用** - 完美适配各种游戏模式
- ✅ **穿透点击不影响操作** - 点击穿透功能，不影响游戏操作
- ✅ **纯覆盖层，不读写游戏内存** - 100% 安全，不会被检测
- ✅ **可调颜色、大小、粗细** - 自定义准星外观
- ✅ **全局快捷键** - 游戏内无需切窗口即可控制
- ✅ **可打包单 exe** - 便携使用，无需安装 Python

## 安装方法

### 方法一：直接使用（需要 Python 环境）

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python main.py
```

### 方法二：打包成 exe（推荐）

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 构建可执行文件：
```bash
python build.py
```

3. 使用生成的 `dist/FPS_Crosshair_Tool.exe`

## 使用说明

### 界面操作

1. **选择预设** - 从下拉菜单选择 200 个预设之一
2. **调整颜色** - 点击"选择颜色"按钮自定义颜色
3. **调整大小** - 拖动大小滑块（5-50像素）
4. **调整粗细** - 拖动粗细滑块（1-10像素）
5. **显示/隐藏** - 点击按钮或使用快捷键
6. **点击穿透** - 勾选后鼠标点击穿透准星

### 快捷键

- `F6` - 显示/隐藏准星
- `F7` - 切换下一个预设
- `F8` - 切换上一个预设
- `Ctrl+Q` - 退出程序

### 准星样式

程序包含以下准星样式：

- **cross** - 十字准星（带间隙）
- **dot** - 点准星
- **circle** - 圆形准星
- **plus** - 加号准星（无间隙）
- **x** - X形准星
- **cross_dot** - 十字+点组合
- **circle_dot** - 圆形+点组合
- **bracket** - 括号准星
- **line** - 单线准星
- **double_line** - 双线准星

## 技术特性

### Windows 专用功能

- 使用 Win32 API 实现窗口穿透
- 支持全屏游戏覆盖
- 系统托盘集成
- 全局热键监听

### 安全性

- 纯覆盖层技术，不修改游戏内存
- 不读取或写入游戏进程
- 不会被反作弊系统检测

### 兼容性

- 支持 Windows 10/11
- 兼容所有主流 FPS 游戏
- 支持全屏和窗口化模式

## 故障排除

### 常见问题

1. **快捷键不响应**
   - 以管理员身份运行程序
   - 检查是否有其他程序占用快捷键

2. **准星不显示**
   - 确保游戏不是独占全屏模式
   - 尝试窗口化或无边框全屏

3. **打包失败**
   - 确保 PyInstaller 版本正确
   - 检查依赖是否完整安装

### 权限要求

- 建议以管理员身份运行
- 需要键盘监听权限
- 需要窗口管理权限
